# 🍲 Food Waste Management Analytics Dashboard

## 📌 Project Overview

The Food Waste Management Analytics Dashboard is designed to analyze food donation, food listing, provider, receiver, and claim data. The project helps identify food distribution patterns, monitor food claims, and provide insights that can reduce food wastage through data-driven decision making.

This project was developed using Python, MySQL, Streamlit, and Plotly for interactive analytics and visualization.

---

## 🎯 Objectives

* Analyze food donation and distribution data.
* Monitor food claims and their status.
* Identify top food providers and receiver categories.
* Track food quantity distribution.
* Reduce food wastage through analytical insights.
* Provide an interactive dashboard for decision-making.

## 🚀 How to Run the Project

### Install Required Libraries

```bash

pip install mysql-connector-python
pip install streamlit
pip install plotly
```

### Run Dashboard

```bash
streamlit run app.py
```
